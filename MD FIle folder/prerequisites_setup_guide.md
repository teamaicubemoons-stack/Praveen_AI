# Prerequisites: Detailed Setup Guide (Zero Se Shuruat)

Agar aapke paas abhi tak koi Business Page ya Instagram account nahi hai, to Meta API setup karne se pehle niche diye gaye steps follow karein.

---

## **Step 1: Facebook Business Page Kaise Banayein?**

Aapka personal account hona zaroori hai, lekin automation ke liye "Page" chahiye hota hai.

1. Apne personal Facebook par login karein.
2. Left side menu mein **"Pages"** par click karein (ya menu icon -> Pages).
3. **"Create New Page"** par click karein.
4. **Page Name**: Apne business ka naam rakhein.
5. **Category**: Apne business ke hisab se select karein (e.g., E-commerce, Clothing Store).
6. **Bio**: Chota sa description likhein.
7. **Create Page** par click karein.
8. Page banne ke baad, profile picture aur cover photo lagayein.

---

## **Step 2: Instagram Business Account Kaise Setup Karein?**

API sirf "Professional" accounts ke liye kaam karti hai.

### **Part A: Account ko Professional mein badlein**

1. Instagram App kholein aur apni **Profile** par jayein.
2. Top right mein **3 lines (Menu)** par click karein.
3. **Settings and Privacy** -> **Account type and tools** mein jayein.
4. **"Switch to Professional Account"** par click karein.
5. **"Business"** select karein (Next, Next karte jayein).
6. Category select karein aur setup complete karein.

### **Part B: Instagram ko Facebook Page se link karein**

1. Instagram Profile mein **"Edit Profile"** par click karein.
2. Niche **"Public Business Information"** section mein **"Page"** par click karein.
3. Wahan se apna banaya hua **Facebook Page** select karke link kar dein.
4. *Alternately*: Aap Facebook Page ki settings mein jaakar "Linked Accounts" se bhi Instagram link kar sakte hain.

---

## **Step 3: WhatsApp Phone Number ki Taiyari**

WhatsApp Cloud API use karne ke liye ek "Saaf" (Clean) number chahiye hota hai.

1. **Naya Number**: Sabse accha hai ki ek naya SIM card use karein jo pehle kabhi WhatsApp par na ho.
2. **Existing Number**: Agar aap apna purana number use karna chahte hain, to aapko us number se **WhatsApp App (Regular ya Business)** ko **Delete/Uninstall** karna hoga.
   * *Dhyan dein*: Cloud API par aane ke baad aap us number ko normal mobile app mein use nahi kar payenge. Saare messages aapke **AI Dashboard/System** par aayenge.
3. **OTP Verification**: Setup ke waqt Meta ek OTP bhejega, isliye woh number active hona chahiye.

---

## **Step 4: Meta Business Suite Setup (Optional but Recommended)**

1. [business.facebook.com](https://business.facebook.com) par jayein.
2. Yahan aapko aapki Page aur Instagram ek saath dikhenge.
3. Yahi se aap future mein **Ad Accounts** aur **Payment Methods** (WhatsApp ke liye) manage karenge.

- [ ] Instagram aur Facebook Page aapas mein link ho gaye hain?
- [ ] WhatsApp ke liye ek number hai jo kisi app mein login nahi hai?
